根据搜索结果，我为你整理了 uv 最新版本的全局配置方法，主要包括包下载镜像源配置和 Python 解释器下载镜像配置两部分：
一、配置文件路径
uv 使用 TOML 格式的配置文件，支持用户级和系统级配置 ：
表格
级别	Linux/macOS 路径	Windows 路径
用户级	~/.config/uv/uv.toml 或 $XDG_CONFIG_HOME/uv/uv.toml	%APPDATA%\uv\uv.toml
系统级	/etc/uv/uv.toml 或 $XDG_CONFIG_DIRS/uv/uv.toml	%SYSTEMDRIVE%\ProgramData\uv\uv.toml
二、包下载镜像源配置（PyPI 镜像）
用于加速 uv add、uv pip install 等命令下载第三方包 ：
配置文件方式（推荐）
创建或编辑 ~/.config/uv/uv.toml（Linux/macOS）或 %APPDATA%\uv\uv.toml（Windows），添加：
toml
复制
[[index]]
url = "https://pypi.tuna.tsinghua.edu.cn/simple/"
default = true
常用国内镜像源
表格
镜像源	URL
清华大学	https://pypi.tuna.tsinghua.edu.cn/simple/
阿里云	https://mirrors.aliyun.com/pypi/simple/
中科大	https://pypi.mirrors.ustc.edu.cn/simple/
豆瓣	https://pypi.douban.com/simple/
华为云	https://mirrors.huaweicloud.com/repository/pypi/simple
环境变量方式（临时或全局）
bash
复制
# Linux/macOS
export UV_DEFAULT_INDEX="https://pypi.tuna.tsinghua.edu.cn/simple"

# Windows PowerShell
$env:UV_DEFAULT_INDEX="https://pypi.tuna.tsinghua.edu.cn/simple"
将上述命令添加到 ~/.bashrc、~/.zshrc 或 Windows 系统环境变量中可永久生效 。
三、Python 解释器下载镜像配置
用于加速 uv python install 下载 Python 版本 ：
配置文件方式
在 uv.toml 中添加：
toml
复制
python-install-mirror = "https://registry.npmmirror.com/-/binary/python-build-standalone/"
或南京大学镜像 ：
toml
复制
python-install-mirror = "https://mirror.nju.edu.cn/github-release/indygreg/python-build-standalone"
环境变量方式
bash
复制
# Linux/macOS
export UV_PYTHON_INSTALL_MIRROR="https://registry.npmmirror.com/-/binary/python-build-standalone/"

# Windows PowerShell
$env:UV_PYTHON_INSTALL_MIRROR="https://registry.npmmirror.com/-/binary/python-build-standalone/"
四、完整配置示例
~/.config/uv/uv.toml 完整示例：
toml
复制
# PyPI 包镜像源
[[index]]
url = "https://pypi.tuna.tsinghua.edu.cn/simple/"
default = true

# Python 解释器下载镜像
python-install-mirror = "https://registry.npmmirror.com/-/binary/python-build-standalone/"
五、验证配置
bash
复制
# 查看当前配置
uv config list

# 测试包下载速度
uv pip install numpy

# 测试 Python 安装
uv python install 3.12 --verbose
六、其他实用工具
如果你不想手动配置，可以使用 cnpip 工具一键测速并自动切换最快镜像源 ：
bash
复制
# 使用 uvx 临时运行（自动配置 uv 镜像源）
uvx cnpip set

# 或显式指定配置 uv
cnpip set --uv
该工具会自动测试各镜像源延迟，选择最快的写入 ~/.config/uv/uv.toml。
