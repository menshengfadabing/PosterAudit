# 品牌合规审核平台 - 前端优化方案

## 一、现状分析

### 当前技术栈
- UI框架：PySide6（原生Qt样式）
- 布局：手动编写QSS样式表
- 组件：原生Qt控件（QPushButton, QLabel, QComboBox等）

### 当前界面问题
1. **视觉风格老旧**：使用基础Qt样式，缺乏现代感
2. **样式不统一**：手动编写的QSS散落在各处，难以维护
3. **交互体验一般**：缺少动画效果和视觉反馈
4. **跨平台一致性差**：不同系统下显示效果不一致

---

## 二、技术选型

### PyQt-Fluent-Widgets 简介

[PyQt-Fluent-Widgets](https://github.com/zhiyiYo/PyQt-Fluent-Widgets) 是一个基于 Fluent Design 设计语言的现代UI组件库。

#### 核心优势
- **Fluent Design 风格**：微软设计语言，现代美观
- **丰富的组件库**：60+ 高质量组件
- **深色/浅色主题**：内置主题切换支持
- **动画效果**：悬停、点击、过渡动画
- **跨平台一致**：Win/Linux/macOS 显示一致
- **活跃维护**：持续更新，文档完善

#### 支持的Qt版本
| Qt版本 | 安装命令 | 分支 |
|--------|----------|------|
| PyQt5 | `pip install PyQt-Fluent-Widgets` | master |
| PyQt6 | `pip install PyQt6-Fluent-Widgets` | PyQt6 |
| **PySide6** | `pip install PySide6-Fluent-Widgets` | **PySide6** |
| PySide2 | `pip install PySide2-Fluent-Widgets` | PySide2 |

> 本项目使用 PySide6，需要安装 `PySide6-Fluent-Widgets`

## 七、代码示例

### 7.1 主窗口基础模板

```python
from PySide6.QtWidgets import QWidget, QHBoxLayout
from PySide6.QtCore import Qt
from qfluentwidgets import (
    FluentWindow, NavigationItemPosition,
    FluentIcon as FIF, setThemeColor
)

class MainWindow(FluentWindow):
    def __init__(self):
        super().__init__()

        # 设置主题色
        setThemeColor('#0078d4')

        # 创建子页面
        self.settingsPage = SettingsPage(self)
        self.auditPage = AuditPage(self)
        self.historyPage = HistoryPage(self)

        # 初始化导航
        self.initNavigation()

        # 窗口设置
        self.setWindowTitle("品牌合规性智能审核平台")
        self.resize(1400, 900)

    def initNavigation(self):
        # 添加导航项
        self.addSubInterface(
            self.settingsPage, FIF.SETTING, "系统设置"
        )
        self.addSubInterface(
            self.auditPage, FIF.DICTIONARY, "设计审核"
        )
        self.addSubInterface(
            self.historyPage, FIF.HISTORY, "报告历史",
            NavigationItemPosition.BOTTOM
        )
```

### 7.2 设置页面模板

```python
from PySide6.QtWidgets import QWidget, QVBoxLayout
from qfluentwidgets import (
    HeaderCardWidget, SettingCard,
    PushSettingCard, ComboBoxSettingCard
)

class SettingsPage(QWidget):
    def __init__(self, parent=None):
        super().__init__(parent)
        self.initUI()

    def initUI(self):
        layout = QVBoxLayout(self)

        # API配置组
        apiGroup = HeaderCardWidget(self)

        self.deepseekKeyCard = SettingCard(

            self
        )
        apiGroup.addWidget(self.deepseekKeyCard)

        # 更多设置项...

        layout.addWidget(apiGroup)
        layout.addStretch()
```

class ResultCard(CardWidget):

## 八、注意事项

### 8.1 兼容性

1. **保留现有业务逻辑**：只替换UI层，不修改服务层代码
2. **渐进式迁移**：可逐页面替换，新旧代码可共存
3. **自定义组件**：ImageDropArea 等自定义组件保留

### 8.2 打包注意事项

需要将 PySide6-Fluent-Widgets 添加到依赖：

```toml
# pyproject.toml
dependencies = [
    # ... 现有依赖
    "PySide6-Fluent-Widgets>=1.11.0",
]
```

build.spec 需要添加：
```python
hiddenimports=[
    # ... 现有导入
    'qfluentwidgets',
]
```

- **非商业项目**：GPLv3 许可证，免费使用
- **商业项目**：需要购买商业许可证
